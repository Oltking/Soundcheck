Ink & the one accent

Warm ink, and a single confident teal

Text is deep espresso, never pure black. Exactly one accent — a refined jade-teal — means live, healthy, performing. It is the only saturated color used decoratively; everything else earns its color from meaning.

Semantic states · shape + color

Severity is a shape first, a color second

No state is ever signalled by color alone. Each carries a distinct glyph so it survives grayscale, color-blindness, and a low-contrast projector.

high severity
Terracotta + diamond. Exploitable now — hard-coded secret, vulnerable dependency.
needs review
Amber + square. Flagged for the Conductor's attention before action.
remediated
Moss + checked circle. Fixed, approved, and shipped in a PR.
on the tape
Sepia + inverted triangle. Superseded ("retakes") kept for audit.
live
Teal + concentric dot. A player actively performing right now.
informational
Slate + bar. Context, not a finding — no action required.
Typography

A crisp grotesk, and a mono that runs the control plane

Inter Tight carries the interface; IBM Plex Mono carries identity — agent handles, control IDs, evidence, diffs, ledger entries. The mono is what sells "this is an operator's instrument." Sentence case everywhere, two weights.

Signature motion

Calm, eased, and meaningful

Motion is reserved for the few moments that build trust: a handoff travelling as a thread of light, a player breathing while it thinks, a finding landing in the Score, the "retake" folding into the archive. Every motion has a reduced-motion fallback.

The thread of light

When one player hands work to another, an eased line travels the path with a soft arrival pulse.

The retake

When a fix is approved the old finding folds into the archive but leaves a thin sepia thread to the Master Tape — visibly "kept for audit." This is the signature moment.
